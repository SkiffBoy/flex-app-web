## 简介

Flex App 是基于 Vue Vben Admin 开发的应用。Forked from [Vue Vben Admin](https://github.com/vbenjs/vue-vben-admin)。创建于 2026 年 6 月 9 日。

## 特性

- **最新技术栈**：使用 Vue3/vite 等前端前沿技术开发
- **TypeScript**：应用程序级 JavaScript 的语言
- **主题**：提供多套主题色彩，可配置自定义主题
- **国际化**：内置完善的国际化方案
- **权限**：内置完善的动态路由权限生成方案

## 安装使用

1. 获取项目代码

```bash
git clone https://github.com/SkiffBoy/flex-app-web.git
```

2. 安装依赖

```bash
cd flex-app-web
npm i -g corepack
pnpm install
```

3. 运行

```bash
pnpm dev
```

4. 打包

```bash
pnpm build
```

非常欢迎你的加入！[提一个 Issue](https://github.com/SkiffBoy/flex-app-web/issues/new/choose) 或者提交一个 Pull Request。

**Pull Request 流程：**

4. 推送您的分支：`git push origin feature/xxxx`
5. 提交 `pull request`

## Git 贡献提交规范

参考 [vue](https://github.com/vuejs/vue/blob/dev/.github/COMMIT_CONVENTION.md) 规范 ([Angular](https://github.com/conventional-changelog/conventional-changelog/tree/master/packages/conventional-changelog-angular))

- `feat` 增加新功能
- `fix` 修复问题/BUG
- `style` 代码风格相关无影响运行结果的
- `perf` 优化/性能提升
- `refactor` 重构
- `revert` 撤销修改
- `test` 测试相关
- `docs` 文档/注释
- `chore` 依赖更新/脚手架配置修改等
- `ci` 持续集成
- `types` 类型定义文件更改
